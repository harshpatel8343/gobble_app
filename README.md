# GOBBLE
 An extensive food recipe archive created by in-house chefs. You can even save your favorite recipes and watch it at your own convenience. You can search among thousands of our recipes and fulfill your cravings.Now that you’ve got all the recipes you need, make sure you have all the essential kitchen tools you need to get cooking!

## Description
• Designed and developed a clean and modern app using Flutter
• Optimized app for searching various food recipes using API integration.
• Utilized responsive design and for database purposes, Firestore is used
